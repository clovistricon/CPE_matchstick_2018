# CPE_matchstick_2018
Jeu du matchstick contre une AI

À exécuter sur le terminal!

Arguments du programme (obligatoires): [Nombre de lignes] [Nombre max de bâtons pouvant être retirés]

exemple:

$CPE_matchstick_2018> ./matchstick 4 3
